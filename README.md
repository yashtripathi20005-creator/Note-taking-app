# 📝 Note-Taking App

A simple, browser-based note-taking application built with Python (Flask) and vanilla JavaScript.

## Features
- ✨ Create, read, update, and delete notes
- 💾 Persistent storage with JSON file
- 🎨 Clean, responsive UI
- ⚡ Real-time updates

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py

# Open browser at http://localhost:5000
